# Lab 00 - Prework

## Objetivo

Dejar el entorno listo para trabajar con Docker, Compose y kubectl.

## Pasos guiados

1. Verificar Docker: `docker --version`
2. Verificar Compose: `docker compose version`
3. Verificar kubectl: `kubectl version --client`
4. Ejecutar contenedor de prueba: `docker run hello-world`

## Comprobacion

Todos los comandos anteriores ejecutan sin error.
