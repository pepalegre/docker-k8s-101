# Lab 01 - Docker Basico

## Objetivo

Entender imagenes, contenedores, volumenes y red en un caso simple.

## Pasos guiados

1. Construir una imagen con `Dockerfile.base`.
2. Ejecutar contenedor local.
3. Montar un volumen para persistir datos.
4. Revisar logs y ciclo de vida (`stop`, `rm`).

## Entregable minimo

Captura o salida de comandos de build, run y persistencia.
